What is Djoshcoin?
----------------

Djoshcoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.
 - 30 second block targets
 - subsidy halves in 4.2M blocks (~4 years)
 - ~21 million total coins
 - 2.5 coins per block
 - 120 blocks to retarget difficulty
